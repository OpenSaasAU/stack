# Search Documentation

Call `opensaas_feature_docs` with a topic to search the OpenSaaS Stack documentation.

Examples:

- `topic: "access control"` - Learn about access control patterns
- `topic: "field types"` - See all available field types
- `topic: "hooks"` - Understand the hooks system
- `topic: "relationships"` - Learn about relationships between lists

This fetches live documentation from https://stack.opensaas.au/ with code examples.
